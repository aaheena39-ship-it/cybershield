# CyberShield - Cyber Crime Complaint Portal

CyberShield is a web-based cyber crime reporting system that allows users to register complaints, upload evidence, and track complaint status.

## Features
- User Registration & Login
- Secure Complaint Submission
- Evidence Upload
- Complaint Tracking
- Admin Dashboard
- Complaint Status Management

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
